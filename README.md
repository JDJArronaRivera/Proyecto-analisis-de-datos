# Portafolio de Análisis de Datos 📊

Este repositorio contiene una colección de proyectos prácticos de Análisis de Datos desarrollados en Python. 

## Proyectos Incluidos:

* **Instacart:** Análisis de patrones de compra y entrega de comestibles.
* **Industria de Videojuegos:** Análisis exploratorio para identificar patrones de éxito comercial en plataformas de gaming.
* **Megaline:** Análisis estadístico para determinar la rentabilidad de las tarifas de telecomunicaciones (Surf vs. Ultimate).

## Herramientas utilizadas:
* Python (Pandas, Matplotlib, Seaborn, SciPy)
* Jupyter Notebooks
